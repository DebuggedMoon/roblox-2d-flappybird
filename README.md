## Deployment
To run the project localy follow the steps described bellow
### Project Setup

```sh
wally install
```

### Compile and Run

```sh
rojo build -o flappybird.rbxlx
```

### Start Server to sync for Hot Reloads 

```sh
rojo serve
```
